# 跆拳道荣誉榜小程序

## 项目介绍

跆拳道荣誉榜小程序是一个基于微信小程序云开发的应用，用于展示跆拳道学员的比赛荣誉记录，提供在线报名功能，以及允许学员查看个人信息和荣誉记录。

## 主要功能

### 1. 荣誉榜（首页）
- 公开展示每个学员获得的比赛荣誉记录
- 按热度值排序显示
- 支持分享功能，每次分享增加一个热度值
- 支持查看荣誉详情

### 2. 报名页面
- 提供电话报名通道
- 支持在线提交报名信息
- 生成报名二维码，方便分享和联系

### 3. 个人中心
- 手机号登录验证身份
- 查看个人基本信息和会费到期信息
- 查看个人荣誉记录
- 支持维护个人基本信息

## 技术架构

本项目采用微信小程序云开发技术栈：

- 前端：微信小程序原生开发
- 后端：云函数
- 数据存储：云数据库
- 文件存储：云存储

### 云函数

- getOpenId: 获取用户OpenID
- getHonorList: 获取荣誉榜列表
- increaseHeat: 增加荣誉热度
- genMpQrcode: 生成小程序码
- submitRegistration: 提交报名信息
- getUserInfo: 获取用户信息
- userLogin: 用户登录
- getUserHonors: 获取用户荣誉
- updateUserInfo: 更新用户信息

### 数据库集合

- honors: 荣誉记录
- students: 学员信息
- registrations: 报名记录

## 部署说明

1. 创建微信小程序项目
2. 开通云开发
3. 创建云环境并获取环境ID
4. 在`app.js`中填入环境ID
5. 上传云函数到云环境
6. 创建数据库集合：honors, students, registrations
7. 设置安全规则
8. 编译运行小程序
